# CALUMAI

CALUMAI 是一個以原生 HTML、CSS 與 JavaScript 製作的族語教學工具作品集，部署於 GitHub Pages。網站採用原創 16-bit 像素視覺，把每項作品整理成可以直接進入的任務關卡。

## 頁面

- `/`：品牌首頁與工具入口
- `/lab/`：依備課、上課、錄音與行政情境整理的工具選單，保留任務看板與公開回饋
- `/blog/`：以存檔介面呈現的製作筆記與文章列表
- `/tools/`：可直接使用的教學與課務工具

## 本機預覽

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

開啟 `http://127.0.0.1:4173/`。

## 設計原則

- 族語內容由人提供與確認，不由 AI 自行翻譯或補寫。
- 工具優先解決真實教學與行政流程。
- 首頁、文章與工具頁都需支援桌面和手機操作。
- 像素素材與介面為原創設計，不使用既有遊戲角色、商標、音效或關卡素材。
