《山林見證》遊戲說明：怎麼玩這款原民法律法庭推理 RPG

《山林見證》是一款原民法律法庭推理 RPG。

你不是進來背法條，也不是只做選擇題。你要做的是聽證詞、讀證據、拆開說法裡太快成立的推論，最後理解一個案件為什麼不能只看表面。

目前版本是公開測試版。它已經可以玩，但法律與文化內容仍需要顧問複核，所以請把它看成一個正在接受檢查的互動作品，不是正式法律意見。

你在遊戲裡要做什麼

每一案都會先給你一段情境。你會看到事件、角色說法、官方或當事人的證詞，以及逐步出現的證據。

你的任務不是急著猜正解，而是判斷：

  • 這句話是在描述事實，還是在做推論？
  • 這份證據真的能支持剛才的說法嗎？
  • 現在還缺哪個關鍵資訊？
  • 哪個結論下得太快？
  • 這個案件最後應該怎麼整理？

如果你玩到一半有一個「等一下，這裡是不是還不能確定」的瞬間，那就是這款遊戲真正想讓你練的能力。

先選身份：律師或法官

遊戲一開始可以選擇兩種身份。

律師模式比較像法庭攻防。你要聽出證詞裡的問題，找出官方說法或對方主張裡沒有被證明的地方，並在適合的時機出示證據。

法官模式比較像判斷整理。你不只看哪一邊講得有道理，還要回到紀錄、證據與法律整理，確認每一步推論是不是站得住腳。

兩種身份看的案件相同，但心態不同。律師比較主動追問，法官比較重視最後的判斷理由。

怎麼讀證詞

證詞不一定等於事實。

角色可能講得很肯定，但他的肯定可能只是因為他只看見一部分。官方紀錄可能看起來很完整，但完整的表格不一定等於完整的理解。當事人可能說得很含糊，但含糊不代表沒有重點。

讀證詞時可以先抓三件事：

  • 他親眼看見了什麼？
  • 他推測了什麼？
  • 他沒有回答什麼？

遊戲裡很多問題不是問你「哪個法律名詞正確」，而是問你這句話中間少了哪一步。

怎麼使用證據

證據會逐步出現，不是所有證據一開始都能拿到。

看到證據時，先不要急著把它當答案。每份證據都要分成兩層來看：

  • 它能證明什麼？
  • 它不能證明什麼？

例如，一份紀錄可以證明某件事被寫進文件，但不一定能證明每個人都真正理解內容。一張照片可以證明某個物品存在，但不一定能證明用途、動機或法律效果。

出示證據時，重點不是把最像答案的東西丟出去，而是確認它剛好能打開目前這個矛盾。

法庭紀錄是你的記憶

案件走到後面，角色說過的話、取得的證據、曾經出現的問題都會變多。這時候不要只靠記憶。

法庭紀錄的功能就是讓你回去檢查：

  • 哪個角色說過什麼？
  • 哪份證據已經取得？
  • 哪個問題還沒有被回答？
  • 你剛才是不是把一份證據看得太滿？

如果你卡住，通常不是因為沒有答案，而是需要回到紀錄裡找出前面漏掉的一步。

信任值不是分數，是提醒

遊戲裡的信任值不是單純的輸贏分數。

它比較像一個提醒：你目前的推理是不是穩，是否太快相信某個說法，或是否忽略了該檢查的證據。

信任值下降時，不一定代表你完全錯了。有時候它是在提醒你：你看到了一個可能方向，但還少了足夠的證據。

判決結果和法律卡怎麼看

每一案結束後，遊戲會給你判決結果與法律整理。

判決結果不是要你背起來，而是回頭看：為什麼前面的某句證詞不能直接成立？為什麼某份證據只能證明一部分？為什麼法院或制度會在這裡特別重視程序、身份、土地、文化權利或知情同意？

法律卡則是結案後的整理。它會把案件背後的法律或制度脈絡拉出來，幫你把剛剛玩的推理過程接回真實世界。

十個案件怎麼理解

目前遊戲有十個案件。它們不是十題考試，也不是全部都對應單一真實法院案件。

比較準確的說法是：這些案件取材自真實判決、法規與制度爭議。

其中有些案件明確參考憲法判決或法院裁判，有些則是從真實法規、行政制度、文化權利或政策爭議改編成遊戲情境。

所以玩遊戲時請記得：角色、場景和部分事件細節是遊戲化改編；真正需要被尊重的是背後的來源、制度問題，以及還不能被過度簡化的文化脈絡。

第一次玩建議怎麼走

第一次玩可以照這個順序：

  1. 先選一個身份，不要一直切換。
  2. 從 Case 01 開始，先熟悉證詞、證據和問題節奏。
  3. 每次看到證據，都先問它能證明什麼、不能證明什麼。
  4. 答錯時不要急著重玩，先看錯誤回饋指出哪一步推論太快。
  5. 結案後再看法律卡，把遊戲裡的推理接回真實來源。

如果你是第一次接觸法律題材，建議不要把目標放在全對。這款遊戲比較想讓你練習的是：知道自己為什麼不該太快確定。

這不是定稿，而是公開測試

《山林見證》目前仍是公開測試版。

它已經能呈現核心玩法：角色證詞、證據閱讀、法庭紀錄、出示證據、判決結果和法律卡。但因為題材牽涉原住民族法律、文化權利、土地、姓名、傳統智慧創作和歷史政策爭議，正式推廣前仍需要法律顧問與適當文化顧問逐案檢查。

所以這篇遊戲說明的目的，不是把作品講成已經完成，而是讓第一次進來的人知道怎麼玩、怎麼讀、怎麼停下來想。

如果你玩完只記得一件事，希望是這句：看起來合理的說法，也要先問它到底被什麼證據支持。
